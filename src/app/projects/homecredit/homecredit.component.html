Methodology

Carried out through steps shown in the Figure 2.

    The steps involved in carrying out this project are:
  • Understanding the business and identifying important issues related to financial loan risks.
  • Performing exploratory data analysis (EDA) on the Home Credit dataset to preprocess the initial data. This helps in understanding the basic information about the Home Credit company and financial business, and identifies the issues that need to be addressed in the next step.
  • Selecting important features from the dataset to predict financial loan risks and processing missing values. Transforming the data into a suitable form for applying machine learning methods and balancing data to ensure that the machine learning model is not biased due to the imbalance of data classes.
  • Dividing the dataset into training and testing sets to evaluate the performance of the machine learning models. Using the Recursive Feature Elimination (RFE) method to select important features and building models using machine learning methods such as Logistic Regression, Random Forest, and XGBoost.
  • Evaluating the performance of the models using evaluation metrics such as confusion matrix and AUC. Selecting the machine learning model with the best performance and using it to select the important features that affect that machine learning method.
  • Predicting the application test set to provide predictions for financial loan risk profiles.
